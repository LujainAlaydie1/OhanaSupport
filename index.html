Privacy Policy

Data Synchronization & Firebase

Ohana uses Google Firebase solely to host the game library and synchronize "Game Night Codes" for event organization. No player names, personal scores, or identifiable data are uploaded to or stored on Firebase servers.

Local Data Only

All game scores, leaderboards, and session timers are processed and stored locally on your device. This data is never transmitted externally.

Third-Party Services

Network connectivity via Firebase is used only to fetch game metadata and images. We do not use any tracking, advertising, or analytics SDKs that identify individual users.
